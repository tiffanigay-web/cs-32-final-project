# cs-32-final-project
CS32 Project with Vito &amp; Emily

This project is called *tentatively* GrowPath! It is inspired by Flora, a productivity app available on the App Store. One deeply personal, customizable, emotionally loaded plant grows with the student user across an entire semester. As students complete assignments, their plant grows; conversely, when students miss assignments and neglect their plant, it begins to wilt. The program is meant to motivate students to complete their work proactively.
